Un Validador JavaScript??

jQValidator es un validador JavaScript basado en la libreria jQuery apuntado a simplificar las validaciones que se hacen en los formularios html, sus principales ventajas son:

Rapido

Diseñamos a este plugin para ser rapido y evitar sobrecargar tu navegador.

No mas codigo!

No ocupas poner mas codigo JS, este validador solo se basa en las clases CSS para hacer su trabajo, mejor aun es dinamico!

Licencia?

Puedes usarlo en tus proyectos personales y de negocios, no hay restriccion, ya que es libre como el aire que respiras.

Flexibilidad

Necesitas validar si y solo si el usuario realizo un hover y despues le pico a... Hey, sabemos que hay clientes dificiles, tambien hay funciones extra e JS para ti ;).

Google Code

todo nuestro progreso y versiones estan en un repositorio GIT de Google Code, si quieres puedes hasta echarnos una mano.

jQuery

Basado en la version 1.9.0 de JQuery lo que significa Cross-Browsing Seguro! olvidate de sufrir por IE.
