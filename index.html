jQValidator

Tavech Js Validator













































Clases que valida el jQValidator por Default
  • Clases de combinacion
    • jqrequired: esta clase se puede usar sola o combinada, hace que el campo con esta clase no se pueda dejar vacio.
    • jqrestrict: esta clase restringe que el usuario no coloque mas chars que los permitidos por la clase validadora que se le combino
    • jqlow: convierte el contenido del input con esta clase en minusculas al hacer un blur
    • jqup: convierte el contenido del input con esta clase en mayusculas al hacer un blur
  • Clases de Formato
    • jqnumber99d99: esta clase se puede usar sola o combinada, hace que el campo con esta clase se ajuste al formato que especificas, ejemplo jqnumber4d3 sera un numero con 4 enteros y 3 decimales.
